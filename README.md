# The classifier for medical pills / Классификатор для медицинских таблеток

### Проект использует pytorch и PIL для распознавания и классификации изображений, DataLoader и kagglehub для загрузки и подготовки данных, sklearn для вычисления метрик

### Проект представляет собой CV классификатор на базе DenseNet для классификации таблеток на фото (белый фон, с тенями, условия съемки хорошие, но очень мало данных).

#### В проекте используются следующие файлы и директории:

- ```pipeline_cv.ipynb``` - тетрадь ipynb с основным пайплайном и объяснениями.
- ```cv_model.py``` - код модели и тренировки для нее, а также оценки точности на test DataLoader
- ```dataset_executor.py``` - все обработчики для данных: загрузка, распаковка в нужное место, создание DataLoader,проверка данных
- ```utils.py``` - служебные функции, напрямую не относящиеся к обучению и данным (используется только установка логгера)
- ```logs/``` -  папка с логами для всех модулей .py (3 штуки)
- ```checkpoints/``` - папка с выгруженными моделями, определенными как лучшие по точности
- ```dataset/``` - папка, в которую загружается датасет
- ```cv_pill_req.txt``` - зависимости проекта. Рассчитаны на Python 3.13.7. Отдельно убраны версии ROCm для корректной работы с Nvidia CUDA.

Порядок работы с проектом:

1. Склонировать репо ```https://github.com/All-C-6/CV_classification_pill.git```
2. Создать окружение ```python -m venv .venv``` (рекомендуется Python 3.13)
3. Запустить все ячейки проекта в тетради ```pipeline_cv.ipynb```, выбрав нужный интерпретатор (из .venv). Все должно скачаться и обработаться автоматически. У меня потребление VRAM было до 5 Гб.
